# CGPA Calculator

A modern, responsive CGPA Calculator built with React and Vite for B.Tech students.

## Features

- **SGPA Calculator** - Calculate Semester Grade Point Average
- **CGPA Calculator** - Calculate Cumulative Grade Point Average
- **Prediction Calculator** - Predict future CGPA based on target grades
- **RTU Syllabus Support** - Built specifically for RTU B.Tech students
- **Dark/Light Theme** - Toggle between dark and light modes
- **Persistent Data** - Saves your data locally

## Tech Stack

- React
- Vite
- Framer Motion
- CSS Variables

## Getting Started

```
bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
